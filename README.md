# Env installation

## Env Windows (Powershell)

```powershell
python -m venv .venv
.venv/Scripts/activate.ps1
python -m pip install -r requirements.txt
```

Maybe this command can help

## Env Unix (Bash)

```bash
python -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
```
